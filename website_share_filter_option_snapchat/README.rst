====================================
Website Share Filter Option Snapchat
====================================

This module merges the functionalities introduced by modules
website_share_snapchat and website_share_filter_option_technical, in order to
introduce a new sharing option (for Snapchat) on website settings. The button
"Share on Snapchat" will thus only be visible on those websites for which it is
allowed.
